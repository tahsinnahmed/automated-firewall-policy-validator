# Automated Firewall Policy Validator
This is a network automation tool to quickly check whether a newly proposed firewall rule already exists in the current policy. If it exists, the requester will not apply for it. If it does not exist, the requester can proceed. This reduces duplicates, speeds up reviews, eliminates human error, and keeps the policy clean.
